ResumeForge is a dynamic resume builder built with React, TypeScript, Tailwind CSS, and shadcn-ui, designed to help users create, preview, and download professional resumes effortlessly.

##  Features
*  Instant Preview: See live changes as you update your information.
*  Dynamic Input System: Add and edit sections like summary, education, and experience.
*  PDF Download:  Export your resume in a neat, professional layout.
*  Modern UI:  Built with Tailwind CSS and shadcn-ui for a sleek look.
*  Responsive Design:  Fully optimized for all devices.
